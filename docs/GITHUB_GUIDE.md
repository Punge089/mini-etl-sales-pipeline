# GitHub Upload Guide

## 1. Create a new repository

Repository name:

```text
mini-etl-sales-pipeline
```

Description:

```text
A Python ETL pipeline project for cleaning, loading, querying, and reporting sales data.
```

## 2. Push the project

Open terminal inside the project folder and run:

```bash
git init
git add .
git commit -m "Initial ETL sales pipeline project"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/mini-etl-sales-pipeline.git
git push -u origin main
```

## 3. Add repository topics

Recommended GitHub topics:

```text
python
pandas
sql
sqlite
etl
data-engineering
data-pipeline
portfolio-project
```

## 4. Before submitting

Check these:
- README is visible on GitHub
- `main.py` is in the root folder
- `src/` folder exists
- `data/raw/sales.csv` exists
- No `venv/` folder uploaded
